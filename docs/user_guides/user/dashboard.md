# User Dashboard Guide

**Last Updated**: 2025-10-24  
**Audience**: End Users  
**Complexity**: Beginner

## Overview

The **User Dashboard** is the main entry point for the IoT Management System user interface. It provides an overview of your devices, recent activity, quick actions, and system status. The dashboard serves as your central hub for monitoring and managing your IoT infrastructure.

## Prerequisites

- **User account** - Valid user account with dashboard access
- **Device access** - Access to at least one IoT device

## Getting Started

### Quick Start
1. **Access Dashboard** - Navigate to the main dashboard
2. **View Overview** - Review device status and system information
3. **Quick Actions** - Use quick action buttons for common tasks
4. **Monitor Activity** - Track recent device activity and events
5. **Navigate Features** - Access different system features from the dashboard

### Navigation
- **Menu Path**: Dashboard (main entry point)
- **URL**: `/user/dashboard`
- **Direct Access**: Click "Dashboard" in the main navigation
