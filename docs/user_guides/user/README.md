## ⏱️ **Operation Timeouts & Logic**

### **Device Operations**
- **Device Restart**: 2 minutes timeout
- **Device Reboot**: 5 minutes timeout
- **Device Shutdown**: 1 minute timeout
- **File Operations**: 5 minutes timeout
- **App Operations**: 10 minutes timeout

### **Bundle Installation**
- **Per App Installation**: 10 minutes timeout
- **Bundle Deployment**: 30 minutes timeout
- **Retry Logic**: 3 attempts per failed app
- **Progress Updates**: Every 30 seconds

### **Profile Application**
- **Per Setting Application**: 3 minutes timeout
- **Profile Deployment**: 15 minutes timeout
- **Retry Logic**: 2 attempts per failed setting
- **Progress Updates**: Every 15 seconds

### **WebRTC Operations**
- **Connection Setup**: 30 seconds timeout
- **Terminal Connection**: 1 minute timeout
- **Remote Desktop**: 2 minutes timeout
- **Screenshot Capture**: 10 seconds timeout
